## Detalhe: chatbot-triagem (fundido de edge-functions.md)

> Original em `docs/_archive/_fundidos/edge-functions.md`. Apenas `chatbot-triagem` estava documentada.
> ⚠️ Este catálogo cobre apenas ~22 de 57 funções — ver TODO de geração em `docs/database-er-diagram.md`.

- **Disparo:** `POST /functions/v1/chatbot-triagem` (`verify_jwt = false`)
- **Entrada:** `{ telefone, nome_perfil, tipo_mensagem: text|audio|image, conteudo_texto, media_id? }`
- **Fluxo:** transcrição Whisper (áudio) → sessão `atendimentos_bot` (24h) → GPT-4o-mini → cria `pedidos_oracao` / `testemunhos` / pedido pastoral → notifica admin
- **Saída:** `{ reply_message, notificar_admin, dados_contato }`
- **Secrets necessários:** `OPENAI_API_KEY`, `WHATSAPP_API_TOKEN`, `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`
